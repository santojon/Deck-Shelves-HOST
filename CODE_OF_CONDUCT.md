# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/),
version 2.1.

In short: be respectful, assume good faith, and keep discussion focused on the
work. Unacceptable behaviour can be reported to the maintainer at the contact in
[SECURITY.md](./SECURITY.md). Maintainers may remove, edit, or reject
contributions that violate this Code of Conduct.
